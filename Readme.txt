SMRT SMART MAPPER (SSM) is an application which helps us with various things under the transportation network.


These include:
• Representation of the network of SMRT stations with an adjacency list
• A user can add Links between two stations
• A user can delete Links between two stations
• Breadth First Search can be visualized
• Depth First Search can be visualized
• Shortest Path between two stations can be found
• A rich and friendly GUI to facilitate all the features is provided.



This Application is developed using PyCharm IDLE since it provides support for the type of application
we are trying to develop.
For serving the Web Server using Flask it is required to install the flask package to the project interpreter
and use python 3.8 .
